# [1439 / 그리디: 뒤집기]

```
n = input()
z = 0
o = 0

if n[0] == '0':
  z += 1
else:
  o += 1

for x in range(len(n) - 1):
  if n[x] != n[x + 1]:
    if n[x + 1] == '0':
      z += 1
    else:
      o += 1

print(min(z, o))
```

입력받은 문자열이 1인지 0인지 분류할 변수를 각각 만들고 첫번째 값을 미리 카운트해둔다. 그 후 반복하면서 현재값과 +1번째의 값을 비교하여 다르다면 0인지 1인지 확인하여 각 카운트에 더한다. 적게 더해진 분류가 최소 뒤집는 횟수이다.
