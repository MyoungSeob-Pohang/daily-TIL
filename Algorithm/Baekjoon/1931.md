# [1026 / 그리디: 회의실 배정]

```
import sys
input = sys.stdin.readline

k = int(input())
n = []
for i in range(k):
  n.append(list(map(int, input().split())))

es = sorted(n, key = lambda x:(x[1], x[0]))

start = es[0][1]
count = 1

for x in range(1, k):
  if es[x][0] >= start:
    count += 1
    start = es[x][1]

print(count)
```

솔직히 정렬하고 제대로 코드 못짜서 시간 굉장히 오래걸렸다. 그래도 정렬하는법 검색해서 알아내고 반례 맞춰서 수십번 실패한 덕에 성공했다.
