# [2828 / 그리디: 사과 담기 게임]

```
n, m = map(int, input().split())
j = int(input())

s = 1
e = m
c = 0

for _ in range(j):
  t = int(input())

  if t < s:
    c += (s - t)
    s = t
    e = (t + m - 1)
  elif t > e:
    c += (t - e)
    e = t
    s = (e - m + 1)

print(c)
```
