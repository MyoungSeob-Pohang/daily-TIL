# [14469 / 그리디: 소가 길을 건너간 이유3]

```
n = int(input())
r = []

for _ in range(n):
  r.append(list(map(int, input().split())))

sr = sorted(r, key = lambda x:(x[0]))

sum = sr[0][0] + sr[0][1]

for x in sr[1:]:
  if sum > x[0]:
    sum += x[1]
  else:
    sum = x[0] + x[1]

print(sum)
```

그렇게 어렵지 않게 풀었던 문제이다. 생각해보면 쉽게 답이 나온다. 정렬을 잘해야한다.
