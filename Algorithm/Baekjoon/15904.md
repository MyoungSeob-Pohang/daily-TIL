# 15904번 : UCPC는 무슨 약자일까

```
target = ['U', 'C', 'P', 'C']
n = input()
r = ''

for x in n:
  if x == 'U' or x == 'C' or x == 'P':
    r += x

i = 0
l = ''

for x in r:
  if l == 'UCPC':
    break
  elif x == target[i]:
    l += x
    i += 1

if l == 'UCPC':
  print('I love UCPC')
else:
  print('I hate UCPC')
```

이번에도 문제를 재대로 읽지 않아서 한참해맸다. 전체를 대문자로 만들어서 푸는 바람에 계속 틀렸다고 나왔다. U인지 C인지 P인지 확인할때 대문자와 소문자는 다르다는 문제를 제대로 읽지못했다.
